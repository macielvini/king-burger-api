## About
This is a personal project for a fictional fast-food.

##### Features
- User registration
- Password encryption
- Login and authentication with JWT
- Find items by category or id

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
